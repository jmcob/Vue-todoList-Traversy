# vue-crash-2021

This is a Todo list app, example of what you can do with Vuejs3 as a starter.

The hole app is from Traversy

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
